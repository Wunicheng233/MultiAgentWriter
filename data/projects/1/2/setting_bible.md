# 《勇者艾因的拼图》设定与大纲

## 一、核心设定
- **书名备选**：《勇者艾因的拼图》《四年后，我们重新讲述勇者》《森林书店的勇者笔记》
- **一句话卖点**：勇者死后四年，女弓箭手走访昔日同伴，用多视角回忆拼出平凡勇者的真实一生，揭开一个所有人都在愧疚中守护的温暖真相。
- **时代背景**：剑与魔法的中古大陆，魔王被击败后四年，大陆恢复和平，通讯靠信使与驿站，魔法只是生活的辅助工具，没有超自然的泛滥应用。

## 二、人物档案
### 主角：希尔维亚
- **身份**：原勇者小队弓箭手，现帝国文献馆编纂员
- **性格**：沉默、敏锐、克制温柔
- **核心目标**：编纂勇者传记，拼出艾因的完整形象，解开自己四年来的心结
- **金手指/能力**：
  - **效果**：过目不忘，能精准记住每个人叙述的细节，快速补全事件拼图
  - **限制**：因过度在意细节，长期被愧疚情绪困扰，不敢直面自己对艾因的感情

### 关键配角（不超过3人）
| 姓名 | 身份 | 与主角关系 | 功能/性格 |
| :--- | :--- | :--- | :--- |
| 雷恩 | 原王国剑圣，现帝国骑士团团长 | 前勇者小队队长，希尔维亚的前上司 | 傲慢外表下藏着自卑，因艾因的死陷入自责，提供勇者性格的第一块反差拼图 |
| 玛格丽特 | 原教会圣女，现地方修道院院长 | 前勇者小队治愈师，希尔维亚的前同伴 | 习惯扮演圣女角色，偷偷爱慕艾因，提供勇者内心柔软的拼图 |
| 奥利弗 | 原皇家学院贤者，现学院首席教授 | 前勇者小队军师，希尔维亚的前同伴 | 天赋论者，被艾因颠覆认知，提供勇者“笨拙创造奇迹”的核心拼图 |
| 塞拉 | 流浪预言者，选中艾因的人 | 勇者之路的引路人，与希尔维亚从未深交 | 揭露艾因成为勇者的真相，完成世界观拼图 |

## 三、世界规则
- **基础规则**：低魔奇幻世界，魔法与武技都需要长期练习才能掌握，天赋决定上限，努力决定下限
- **通用限制**：无特殊超自然规则，所有能力都遵循“付出多少收获多少”的基础逻辑

## 四、分章大纲

**重要规则：每章目标字数是 `2000` 字，请根据这个字数规划本章情节容量：**
- 字数越多，每章需要安排越丰富的情节内容（更多细节描写、更多情节推进、更多场景转换）
- 字数越少，情节越紧凑，聚焦核心冲突

| 章节 | 本章目标（情节推进） | 核心冲突/爽点 | 结尾钩子 |
| :--- | :--- | :--- | :--- |

| 第2章 | 希尔维亚前往修道院拜访玛格丽特，玛格丽特接受访谈 | 玛格丽特讲述艾因深夜偷偷练习治愈魔法到双手流血的往事，展现艾因不为人知的脆弱 | 玛格丽特拿出一条沾着旧血的绷带，说是艾因当年留给她的，进一步加深勇者之死的谜团 |
| 第3章 | 希尔维亚前往皇家学院拜访贤者奥利弗，奥利弗勉强接受访谈 | 奥利弗讲述艾因用三个月时间笨方法破解魔王城迷宫的往事，颠覆了他“天赋决定一切”的认知 | 奥利弗拿出艾因画的迷宫草图，上面布满了密密麻麻的标记，他说“艾因的死，我一直有责任” |
| 第4章 | 希尔维亚整理三份访谈材料，触发自己的回忆，补全前三件事的侧面 | 希尔维亚回忆自己当年默默观察艾因的细节：她看到雷恩在艾因练习时偷偷指点，看到玛格丽特每天偷偷给艾因换绷带，看到奥利弗悄悄帮艾因修正迷宫标记 | 希尔维亚翻出自己四年来一直保留的、艾因帮她找的射箭位置标记图，第一次意识到自己对艾因的感情超出同伴 |

| 第7章（尾声1） | 同伴们解开彼此的愧疚，承认艾因的选择，重新接受了勇者死亡的事实 | 四个人坐在当年和艾因一起露营的草地上，第一次轻松地谈起艾因的糗事 | 雷恩说“他从来没怪过我们”，所有人都沉默着笑了 |
